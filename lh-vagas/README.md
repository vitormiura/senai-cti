# LH Vagas 

Projeto simples voltado para o estudo de Angular e API REST como atividade do Senai. Estilização simples apenas para a exibição dos dados utilizando o framework Boostrap.

## Como rodar?

Digite `ng serve` no terminal e acesse o `http://localhost:4200/`. Após isso digite no terminal `json-server --watch vagas-db.json` para simular o BD usando este arquivo JSON.
